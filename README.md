
# Styled-Ui-Kit

StyledUiKit is a React UI component library
built with [styled-components][sc].

## Installed

```sh
npm i --save styled-ui-kit
```
## Getting Started

```jsx
import React from 'react'
import { Theme, configTheme, globalConfig } from 'styled-ui-kit'

globalConfig();

const App = props => (
  <Theme>
      Hello World
  </Theme>
)
```
